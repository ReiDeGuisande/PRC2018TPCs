Resumo

Este trabalho é referente à extensão de uma ontologia, acerca da tabela periódica. É pedido que se acrescente à estrutura existente o novo conceito de molécula. Para tal, foi criada recorrendo ao Protégé a classe molécula, e as Data Properties necessárias, para caracterizar os seus indivíduos. Para além disso, foi criada uma Object Property "hasElement" que relaciona as moléculas, com os elementos que as compõem. Posteriormente, foi feito também um povoamento mínimo, com as moléculas usadas no exemplo do enunciado.
